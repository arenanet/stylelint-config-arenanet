Shareable [StyleLint]() config for enforcing ArenaNet CSS standards.

## Install

```shell
npm i stylelint-config-arenanet
```

## Usage

```js
// package.json
{
    // ...
    "stylelint": {
        "extends": "stylelint-config-arenanet"
    }
}
```
